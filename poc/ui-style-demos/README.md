
# OpenWeave UI Style Demos

This directory contains four standalone static HTML demos for comparing candidate UI directions for OpenWeave.

## Files
- `index.html` - entry page linking to all demos
- `graphite-ops.html`
- `blueprint-canvas.html`
- `slate-terminal.html`
- `paper-rail.html`
- `shared.css` - common shell/layout primitives and theme variables
- `previews/*.png` - generated preview screenshots

## Review method
Open `index.html` and then open each theme page in a browser. Every page keeps the same product scenario:
- top shell
- workspace rail
- infinite canvas with note / terminal / portal / file-tree nodes
- inspector panel
- run drawer

That keeps the comparison focused on visual system choices rather than feature differences.
